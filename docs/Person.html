Person implement Thing

object

A person (alive, dead, undead, or fictional).

Properties

spouse

Type Person

The person's spouse.

additionalName

Type string

An additional name for a Person, can be used for a middle name.

hasCredential

Type EducationalOccupationalCredential

A credential awarded to the Person or Organization.

telephone

Type string

The telephone number.

honorificPrefix

Type string

An honorific prefix preceding a Person's name such as Dr/Mrs/Mr.

parents

Type Person

A parents of the person.

familyName

Type string

Family name. In the U.S., the last name of a Person.

givenName

Type string

Given name. In the U.S., the first name of a Person.

knowsAbout

Type string, string, Thing

Of a [[Person]], and less typically of an [[Organization]], to indicate a topic that is known about - suggesting possible expertise but not implying it. We do not distinguish skill levels here, or relate this to educational content, events, objectives or [[JobPosting]] descriptions.

performerIn

Type Event

Event that this person is a performer or participant in.

netWorth

Type MonetaryAmount, PriceSpecification

The total financial value of the person as calculated by subtracting assets from liabilities.

award

Type string

An award won by or for this item.

workLocation

Type Place, ContactPoint

A contact location for a person's place of work.

honorificSuffix

Type string

An honorific suffix following a Person's name such as M.D. /PhD/MSCSW.

deathPlace

Type Place

The place where the person died.

homeLocation

Type Place, ContactPoint

A contact location for a person's residence.

funding

Type Grant

A [[Grant]] that directly or indirectly provide funding or sponsorship for this item. See also [[ownershipFundingInfo]].

vatID

Type string

The Value-added Tax ID of the organization or person.

height

Type Distance, QuantitativeValue

The height of the item.

globalLocationNumber

Type string

The [Global Location Number](http://www.gs1.org/gln) (GLN, sometimes also referred to as International Location Number or ILN) of the respective organization, person, or place. The GLN is a 13-digit number used to identify parties and physical locations.

contactPoints

Type ContactPoint

A contact point for a person or organization.

hasOccupation

Type Occupation

The Person's occupation. For past professions, use Role for expressing dates.

gender

Type GenderType, string

Gender of something, typically a [[Person]], but possibly also fictional characters, animals, etc. While https://schema.org/Male and https://schema.org/Female may be used, text strings are also acceptable for people who do not identify as a binary gender. The [[gender]] property can also be used in an extended sense to cover e.g. the gender of sports teams. As with the gender of individuals, we do not try to enumerate all possibilities. A mixed-gender [[SportsTeam]] can be indicated with a text value of "Mixed".

parent

Type Person

A parent of this person.

awards

Type string

Awards won by or for this item.

sibling

Type Person

A sibling of the person.

jobTitle

Type string, DefinedTerm

The job title of the person (for example, Financial Manager).

birthDate

Type date

Date of birth.

funder

Type Organization, Person

A person or organization that supports (sponsors) something through some kind of financial contribution.

makesOffer

Type Offer

A pointer to products or services offered by the organization or person.

children

Type Person

A child of the person.

worksFor

Type Organization

Organizations that the person works for.

knows

Type Person

The most generic bi-directional social/work relation.

interactionStatistic

Type InteractionCounter

The number of interactions for the CreativeWork using the WebSite or SoftwareApplication. The most specific child type of InteractionCounter should be used.

address

Type string, PostalAddress

Physical address of the item.

memberOf

Type ProgramMembership, Organization

An Organization (or ProgramMembership) to which this Person or Organization belongs.

publishingPrinciples

Type string, CreativeWork

The publishingPrinciples property indicates (typically via [[URL]]) a document describing the editorial principles of an [[Organization]] (or individual e.g. a [[Person]] writing a blog) that relate to their activities as a publisher, e.g. ethics or diversity policies. When applied to a [[CreativeWork]] (e.g. [[NewsArticle]]) the principles are those of the party primarily responsible for the creation of the [[CreativeWork]]. While such policies are most typically expressed in natural language, sometimes related information (e.g. indicating a [[funder]]) can be expressed using schema.org terminology.

colleagues

Type Person

A colleague of the person.

deathDate

Type date

Date of death.

email

Type string

Email address.

affiliation

Type Organization

An organization that this person is affiliated with. For example, a school/university, a club, or a team.

siblings

Type Person

A sibling of the person.

birthPlace

Type Place

The place where the person was born.

brand

Type Organization, Brand

The brand(s) associated with a product or service, or the brand(s) maintained by an organization or business person.

sponsor

Type Organization, Person

A person or organization that supports a thing through a pledge, promise, or financial contribution. e.g. a sponsor of a Medical Study or a corporate sponsor of an event.

naics

Type string

The North American Industry Classification System (NAICS) code for a particular organization or business person.

contactPoint

Type ContactPoint

A contact point for a person or organization.

callSign

Type string

A [callsign](https://en.wikipedia.org/wiki/Call_sign), as used in broadcasting and radio communications to identify people, radio and TV stations, or vehicles.

isicV4

Type string

The International Standard of Industrial Classification of All Economic Activities (ISIC), Revision 4 code for a particular organization, business person, or place.

hasPOS

Type Place

Points-of-Sales operated by the organization or person.

alumniOf

Type EducationalOrganization, Organization

An organization that the person is an alumni of.

duns

Type string

The Dun & Bradstreet DUNS number for identifying an organization or business person.

weight

Type QuantitativeValue

The weight of the product or person.
